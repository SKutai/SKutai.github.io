Video Amplification

Ursinus REU 2021

Magnifying Minute Motion in 2D and 3D

If you point a camera at a person trying to sit still, the camera can pick up on small motions like the person's breathing and pulse. In this project, my RUE team and I set out to capture such motion and amplify it so that a person can view what would otherwise be hidden from them. When we are finished, we will be able to capture and amplify motion in higher dimesions.

Here is a link to one of the papers that were an inspiration for the project: Eulerian Video Magnification for Revealing Subtle Changes in the World

Ray Tracer

CS-476

Computer Graphics

Add real time raytracing to a scene.

In this assignment I used GLSL and JavaScript to add raytracing to a scene. The ray tracer works in real time and everything can be viewed in a browser. It allows for different effects to be visible such as the reflections of light in mirrors, the shininess of the objects, and the shadows that are cast when a light ray is blocked. In order to build up the ray tracer, I was required to complete a number of tasks like add Blinn-Phong lighting, find the closest surface to the camera that a ray hit, and also calculate for the distortions of objects that were stretched or rotated. Polygon Mesh Processing

CS-476

Computer Graphics

Discover desired details within a mesh.

For this, I needed to implement a list of mesh processing methods in order to view details about various half edge meshes. The methods were programmed using JavaScript; and finished project allows a user to do things like highlight the edges that are related to a face or a point, or find the area and normal of a face. Along with this, one can also inflate and deflate a mesh or smooth and sharpen the mesh as much as they want to. The image to the left shows off the normal of the points of the mesh (the purple lines), the edges related to the one of the points (the blue lines), and the edges to the mesh that are related to a null face (the green line).

Safari Man

CS-474

Human Computer Interactions

Allow a user to search the internet by voice.

This was a project from when I studied human computer interactions. I and my other group members worked used python to make an interactive app that takes speech as input. The program allows users to log in using a username and password or use neither and continue as a guest. After that, users can interact with the computer in a number of ways and have the program do things like open google chrome to make a google search. When finished, a person can say “goodbye” and the program will close itself.

Location Note

CS-474

Human Computer Interactions

Save notes to a real location.

Location Note is the name that I gave to the app that I made in my human computer interactions class. It is an android app that allows users to save a note to an address. Then when the person visits that location, they will get a notification that has the note that they wrote. The app also saves the notes that people have made so that they can view them at any time.

Summary of Project
Ray Tracer

CS-476

Add real time raytracing to a scene.

In this assignment I used GLSL and javascript to add raytracing to a scene. The raytracer works in real time and everythig can be viewed in a browser. It allows for different effects to be visible such as the reflections of objects in a flat, curved, or stretched mirror.

Polygon Mesh Processing

CS-476

Discover desired details within a mesh.

This assignment required that I implement a list of mesh processing methods.

Scene Trees

CS-476

Build a scene of transformed objects and scense.

For this I was tasked with building different scenes by manualy transforming objects and other scenes to get desired results.

Safari Man

CS-474

Allow a user to search the internet by voice.

This is from when I studied human computer interactions. The program takes the user's voice as the only form of input. It starts by allowing a person to log in with a username and password, or by having the user make an acount. They can then make searches for information and when the system gives them a response it will also save their search history.

Location Note

CS-474

Save notes to a real location.

This is an app that allows users to make a note and then save the note to a specific location. Later on when the person visits the spceified location they are given a notification on their phone.

Smoke Stack

CS-476

Create virtual version of a muralist's work.

Recently a muralist repainted an old smoke stack at Ursinus College. The goal of this ongoing project is to create an online virtual representation of their work for other people to view.

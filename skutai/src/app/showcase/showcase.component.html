Fake Cats
The mission of this assignment was to generate fake cat photos. The project was done with tensorflow in python. The method that was used to generate the photos was a convolutional Neural Network. The network can be used to accept many kinds of similar images and then generate new versions of the same thing that could possibly fool a human.
